[Cursor 사용 AI Agent 제작 실습 질의문]

프로젝트 폴더명 : AI_Agent_Basic_Project

Cursor에서 프로젝트 폴더를 연 다음 Ctrl + Shift + P를 눌러 
Python : Select Interpreter에서 anaconda3에서 사용했던 Python 3.12.7을 선택해준다

[질의문]
----------------------------------------------------------------------------------------------------------
최신 LangChain v1.0, OpenAI API(gpt-4o-mini) 기반으로 아래 요구사항을 충족하는 AI 에이전트 프로젝트 코드를 생성하라

[프로젝트 목표]
이미지 분석 +  RAG 검색 + 에이전트 도구 실행이 통합된 멀티모달 AI 에이전트 챗봇을 구현한다.

[사용 기술]
- Python 3.11 이상
- LangChain v1.0 이상 (create_agent 기반)
- langchain-openai
- OpenAI gpt-4o-mini
- 이미지 분석
- 간단한 RAG(문서 3~5개 샘플 저장 후 검색 가능)
- Streamlit 기반 웹 UI

[필수 기능]
1) 이미지 업로드 → 이미지 내용 분석(설명, 캡션 생성 생성)
2) RAG 기반 문서 검색 기능 포함
3) 몇 개의 Tool을 가진 에이전트 생성
   - summarize_doc(text): 문서를 요약한다.
   - extract_keywords(text): 핵심 키워드를 추출한다.
   - search_docs(query): RAG 기반 문서 검색
4) 유저 입력(텍스트/음성/이미지)에 따라 자동으로 적절한 Tool을 호출
5) 마지막 응답은 result["messages"][-1].content 형태로 출력 가능하게 설계

[폴더 구조 예시]
project/

[생성해야 하는 코드 목록]
- requirements.txt
- RAG 임베딩 초기화 및 검색 코드
- Tool 3개 구현 코드
- create_agent()로 멀티모달 에이전트 생성 코드
- 이미지 분석 함수(caption)
- Streamlit UI 코드(app.py):  
  이미지 업로드/텍스트 입력 → 에이전트 실행 → 결과 출력

[주요 조건]

- create_agent 사용
