[PHP Web Server 프로젝트 실습 질의]

프로젝트 폴더명 : PHP_Basic_Project

실습 목표 : PHP 기본 문법과 웹 처리 흐름을 단계별로 구현하며 동작을 이해

실행 환경 : 로컬 PHP 내장 서버 또는 웹호스팅(Apache/PHP) 환경

-------------------------------------------------
PHP 용 툴 설치 : XMAMPP 설치 (가장 쉬움)
https://www.apachefriends.org/download.html 에서 XAMPP 다운로드(144MB)
설치 후 XAMPP Control Panel에서 Apache 시작

PHP 실행 방법 : php_app 폴더의 소스를 C:\xampp\htdocs\php_app에 복사

(C:\xampp\htdocs\ 아래에 php_app 폴더를 미리 만들어 놓고 어래 명령 실행)
Copy-Item -Path .\php_app\* -Destination 'C:\xampp\htdocs\php_app\' -Recurse -Force

브라우저에서 http://localhost/php_app/index.php 접속
-------------------------------------------------


#1단계 : 프로젝트 초기 구조 작성

PHP로 동작하는 단계별 실습 프로젝트를 만들어줘.
소스 내용은 없이 폴더 구조만 아래처럼 만들어줘:

/php_app/
 ┣ index.php
 ┣ hello.php
 ┣ form.php
 ┣ form_process.php
 ┣ calc.php
 ┣ upload.php
 ┣ upload_process.php
 ┣ session_start.php
 ┣ session_page.php
 ┣ logout.php
 ┣ cookie_set.php
 ┣ cookie_read.php
 ┣ style.css
 ┗ README.md

각 파일은 비어 있어도 된다.


#2단계 : 메인 페이지(index.php) 구현

index.php 파일을 작성해줘.

요구사항:
- 실습 메뉴를 링크 목록으로 만들어줘
- 링크 대상은 hello.php, form.php, calc.php, upload.php, session_start.php, cookie_set.php 이다
- 페이지 상단에 "PHP 기능 단계별 실습" 제목을 표시해줘
- style.css를 연결해줘


#3단계 : PHP 기본 출력(hello.php) 구현

hello.php 파일을 작성해줘.

요구사항:
- PHP로 현재 날짜/시간을 출력해줘
- 변수 2개를 선언하고 문자열로 합쳐 출력해줘
- 간단한 조건문(if) 예시를 출력해줘
- index.php로 돌아가는 링크를 넣어줘


#4단계 : 폼 입력 화면(form.php) 구현

form.php 파일을 작성해줘.

요구사항:
- name, email, age 입력 폼을 만들어줘
- method는 POST로 해줘
- action은 form_process.php로 보내줘
- index.php로 돌아가는 링크를 넣어줘


#5단계 : 폼 처리(form_process.php) 구현

form_process.php 파일을 작성해줘.

요구사항:
- POST로 전달된 name, email, age를 받아서 화면에 출력해줘
- name과 email은 필수로 검증해줘(없으면 에러 메시지)
- age는 숫자인지 검증해줘(없으면 null 허용)
- 입력값을 안전하게 출력하기 위해 htmlspecialchars를 적용해줘
- form.php로 다시 돌아가는 링크를 넣어줘


#6단계 : 계산기(calc.php) 구현

calc.php 파일을 작성해줘.

요구사항:
- 두 숫자(num1, num2)와 연산자(op: +, -, *, /)를 입력받는 폼을 만들어줘
- 한 페이지에서 입력과 결과가 같이 보이게 해줘(self submit)
- 나누기에서 0으로 나누는 경우 에러 메시지를 보여줘
- index.php로 돌아가는 링크를 넣어줘


#7단계 : 파일 업로드 화면(upload.php) 구현

upload.php 파일을 작성해줘.

요구사항:
- 단일 파일 업로드 폼을 만들어줘
- enctype은 multipart/form-data로 설정해줘
- action은 upload_process.php로 보내줘
- 업로드 허용 확장자는 jpg, png, pdf로 제한하는 안내 문구를 넣어줘
- index.php로 돌아가는 링크를 넣어줘


#8단계 : 파일 업로드 처리(upload_process.php) 구현

upload_process.php 파일을 작성해줘.

요구사항:
- 업로드 폴더 uploads/ 를 사용하고 없으면 자동 생성해줘
- 업로드된 파일의 확장자를 검사해 jpg/png/pdf만 허용해줘
- 파일명이 겹치지 않게 저장되게 해줘(시간 또는 랜덤값 사용)
- 업로드 성공 시 저장된 경로를 출력해줘
- 실패 시 에러 메시지를 출력해줘
- upload.php로 돌아가는 링크를 넣어줘


#9단계 : 세션 시작(session_start.php) 구현

session_start.php 파일을 작성해줘.

요구사항:
- session_start()를 사용해 세션을 시작해줘
- 간단한 로그인 시뮬레이션을 만들어줘
  입력: username
  버튼: "로그인"
- 로그인하면 세션에 username을 저장하고 session_page.php로 이동시키는 방식으로 만들어줘
- index.php로 돌아가는 링크를 넣어줘


#10단계 : 세션 확인 페이지(session_page.php) 구현

session_page.php 파일을 작성해줘.

요구사항:
- 세션에서 username을 읽어 환영 메시지를 출력해줘
- 로그인 안 된 경우 session_start.php로 이동시키는 안내를 보여줘
- "로그아웃" 링크(logout.php)를 제공해줘
- index.php로 돌아가는 링크를 넣어줘


#11단계 : 로그아웃(logout.php) 구현

logout.php 파일을 작성해줘.

요구사항:
- 세션을 종료해줘(session_unset, session_destroy)
- 로그아웃 완료 메시지를 출력해줘
- session_start.php 또는 index.php로 이동할 수 있는 링크를 넣어줘


#12단계 : 쿠키 저장(cookie_set.php) 구현

cookie_set.php 파일을 작성해줘.

요구사항:
- name 쿠키를 저장하는 폼을 만들어줘
- 쿠키 유효기간은 1일로 설정해줘
- 저장 후 cookie_read.php로 이동하거나 링크로 이동하게 해줘
- index.php로 돌아가는 링크를 넣어줘


#13단계 : 쿠키 읽기(cookie_read.php) 구현

cookie_read.php 파일을 작성해줘.

요구사항:
- 저장된 쿠키(name)를 읽어 화면에 출력해줘
- 쿠키가 없으면 안내 메시지를 출력해줘
- cookie_set.php로 돌아가는 링크를 넣어줘


#14단계 : 공통 CSS(style.css) 작성

style.css 파일을 작성해줘.

요구사항:
- 페이지 폭을 제한하고 가운데 정렬해줘
- 메뉴 링크를 보기 좋게 리스트 형태로 만들어줘
- 버튼 hover 효과를 넣어줘
- 기본 폰트와 여백을 정리해줘
- 웹페이지의 색상을 세련되게 수정해줘

(오류시 입력 : css 파일이 전혀 동작하지 않아)


#15단계 : 실행 방법(README.md) 작성

README.md 파일을 작성해줘.

요구사항:
- 로컬에서 실행하는 방법을 작성해줘
  예: php -S localhost:8000 -t php_app
- 브라우저에서 접속할 주소를 작성해줘
- 각 실습 파일의 목적을 1줄씩 요약해줘
- 자주 발생하는 오류 2가지와 해결 방법을 작성해줘
- php 개발 환경을 위해서 아래와 같은 설치 및 소스 실행 내용도 정리해줘
