[OpenAI 기반 멀티모달 챗봇 구현 실습 질의문]

프로젝트명 폴더명 :  OpenAI_MultiModal_Chatbot_Project

-----------------------------------------------------------------------------
#1단계: 프로젝트 개요 및 목표 정의

질의문:
OpenAI API의 멀티모달 기능을 활용하여
웹캠으로 촬영한 얼굴 이미지를 분석하고
얼굴 표정(감정 상태)을 텍스트로 설명해주는
멀티모달 챗봇 실습 프로젝트를 설계해줘.

다음 조건을 만족하는 기술 스택과 전체 아키텍처를 설계해줘.
아직 소스코드는 구현하지 말고 폴더 구조만 파일들만 만들어줘

- Python 기반
- Streamlit으로 웹 UI 구성
- OpenAI Vision 모델 사용
- 웹캠으로 얼굴 이미지 촬영
- 촬영된 이미지를 OpenAI에 전달
- 분석 결과를 화면에 출력


#2단계: 개발 환경 및 필수 라이브러리 설정

질의문:
OpenAI Vision 기반 얼굴 표정 분석 챗봇을 만들기 위해
필요한 Python 라이브러리 목록과 설치 명령을 작성해줘.

requirements.txt 형태로 정리해주고
각 라이브러리의 역할을 간단히 설명해줘.

#3단계: Streamlit 웹 UI 기본 구성

질의문:
Streamlit을 사용하여 다음 기능을 가진 기본 웹 화면을 구현해줘.

- 웹페이지 제목 표시
- 웹캠(카메라) 활성화
- 사진 촬영 버튼
- 촬영된 이미지 미리보기 영역

아직 OpenAI 연동은 하지 말고
UI 중심으로 app.py 예제를 만들어줘.

#4단계: 카메라 이미지 캡처 및 파일 처리

질의문:
Streamlit에서 웹캠으로 촬영한 이미지를
PIL 또는 bytes 형태로 처리하는 코드를 구현해줘.

- 촬영 버튼 클릭 시 이미지 캡처
- OpenAI API로 전송 가능한 형태로 변환
- 임시 파일 또는 메모리 처리 방식 설명 포함

#5단계: OpenAI Vision API 연동

질의문:
촬영된 얼굴 이미지를 OpenAI Vision 모델로 전달하여
이미지 분석 요청을 보내는 코드를 작성해줘.

조건:
- 최신 OpenAI Python SDK 사용
- 이미지 + 텍스트 프롬프트를 함께 전달
- 얼굴 표정 분석에 초점을 둔 요청 형태

#6단계: 얼굴 표정(감정) 분석 프롬프트 설계

질의문:
얼굴 이미지를 분석하여 표정을 판단하기 위한
OpenAI Vision 프롬프트를 작성해줘.

다음 감정 범주 중 하나 이상을 추론하도록 해줘.
- 행복
- 슬픔
- 분노
- 놀람
- 중립
- 긴장/불안

추론 근거를 짧게 설명하도록 프롬프트를 설계해줘.

#7단계: OpenAI 응답 결과 파싱 및 출력

질의문:
OpenAI Vision API의 응답 결과를 파싱하여
Streamlit 화면에 다음 형식으로 출력해줘.

- 추정된 얼굴 표정
- 간단한 설명 문장

예외 상황(분석 실패, 얼굴 인식 불가) 처리 로직도 포함해줘.


#8단계: 전체 코드 통합 (완성형 app.py)

질의문:
지금까지 만든 내용을 모두 통합하여
실행 가능한 완성형 Streamlit 앱(app.py)을 만들어줘.

- 웹캠 촬영
- OpenAI Vision 호출
- 얼굴 표정 분석
- 결과 출력

(오류 수정: gpt-4o 모델이 표정으로 감정 분석이 어려우니 최신에 출시된 gpt-5.2 모델을 사용해줘
https://platform.openai.com/docs/pricing
gpt-5.2 모델은 코드 스타일이 달라졌어. 오류 수정해줘)

(추가 질의: gpt-4o 모델과 gpt-5.2 모델의 코드 스타일이 달라진 내용을 .md 문서로 만들어줘)
----------------------------------------------------------------------------
#9단계: 확장 실습 과제 제시

질의문:
이번 얼굴 표정 분석 멀티모달 챗봇을 확장할 수 있는
추가 실습 과제를 3가지 제안해줘.

예:
- 표정 기반 상태 메시지 생성
- 여러 사람 얼굴 분석
- 결과 로그 저장
